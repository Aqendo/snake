# My very snake-game
> Intended to be able to run on any POSIX-compatible systems
## Features
- Snake
- Game
- Fun
- Own implementation
- Supports apples
- Supports collisions
- Supports teleporting from one side to another
- Terminal-based
- Terminal is BASED
- No external libraries needed
- Just one file
- ~200 lines
- Handles CTRL-C

## Compiling
Ther is a CMakeLists.txt but you can compile it with just 1 command:
```console
$ g++ snake.cpp -o snake
$ ./snake
```
